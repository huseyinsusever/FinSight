# FinSight
Financial Analysis and Stock Price Prediction using Python and Machine Learning
# FinSight

## Türkçe Açıklama
FinSight, GDP, enflasyon ve faiz verilerini kullanarak hisse fiyatlarını tahmin eden ve finansal trendleri görselleştiren bir web uygulamasıdır. Kullanıcılar faiz oranı, enflasyon ve GDP büyüme oranını girerek hisse tahminlerini görebilir ve interaktif grafiklerle trendleri inceleyebilir.

### Özellikler
- GDP ve finansal veri tahmini
- Kullanıcı girişi ile interaktif hesaplamalar
- Grafiklerle görselleştirme
- Flask tabanlı web arayüzü

---

## English Description
FinSight is a web application that predicts stock prices using GDP, inflation, and interest rate data, and visualizes financial trends. Users can input interest rate, inflation, and GDP growth to see stock price predictions and analyze trends interactively.

### Features
- GDP and financial data forecasting
- Interactive user input for predictions
- Visualization of trends with charts
- Flask-based web interface
